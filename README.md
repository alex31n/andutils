### Install

To get a Git project into your project:

&nbsp;
#### Gradle 

Sept 1: Add it in your root build.gradle at the end of repositories:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

&nbsp;
&nbsp;
Step 2. Add the dependency
```
	dependencies {
		implementation 'com.github.User:Repo:Tag'
	}
```
